## 배운 점 & 아쉬운 점

배운점<br>
섹션에 높이값 설정후 스크롤 모션을 통한 퍼센트 적용하여, 텍스트 노출하는 법을 알았습니다.

아쉬운점<br>
처음 접속시 양쪽에서 나오는 화면을 좀 더 다채롭게 만들지 못해서 아쉽습니다.
